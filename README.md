# 🏏 IPL 2026 Analytics & IPL 2027 Prediction Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard built using IPL 2026 match data. The dashboard provides insights into team performance, player statistics, venue analysis, and a prediction page for IPL 2027 based on historical performance trends.

The goal of this project is to demonstrate data analysis, data visualization, and dashboard development skills using Power BI.

---

## 🚀 Features

### Team Performance Analysis

* Total wins by each team
* Team ranking comparison
* Team-wise performance visualization

### Player Analysis

* Top run scorers
* Player of the Match analysis
* Player impact comparison

### Venue Analysis

* Matches played at each venue
* Venue-wise winning trends
* Stadium performance insights

### Match Analysis

* Match count analysis
* Winning team distribution
* Season overview statistics

### IPL 2027 Prediction Dashboard

* Predicted Winner
* Predicted Runner-Up
* Predicted Playoff Teams
* Team Strength Comparison
* Winning Probability Analysis

---

## 📊 Dashboard Pages

### 1. Team Performance Dashboard

Analyzes the performance of IPL teams based on wins and rankings.

### 2. Player Performance Dashboard

Highlights top scorers and Player of the Match performers.

### 3. Venue Analysis Dashboard

Provides venue-wise insights and match distribution.

### 4. Match Explorer Dashboard

Interactive analysis of match results and team performance.

### 5. IPL 2027 Prediction Dashboard

Forecasts potential IPL 2027 outcomes based on IPL 2026 data.

## 📈 Key Insights

* RCB emerged as the strongest team in IPL 2026.
* GT, RR, and SRH were among the top-performing teams.
* Mitchell Marsh was one of the leading run scorers.
* Multiple venues hosted competitive matches throughout the season.
* IPL 2027 predictions indicate RCB and GT as strong title contenders.
